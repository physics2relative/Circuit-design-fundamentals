# Glossary

## Digital

- **RTL**: Register Transfer Level. 레지스터 사이 데이터 이동과 조합논리를 기술하는 설계 수준입니다.
- **FSM**: Finite State Machine. 상태와 전이 조건으로 동작을 표현하는 순차논리 모델입니다.
- **Setup Time**: 클럭 엣지 전에 데이터가 안정적으로 유지되어야 하는 최소 시간입니다.
- **Hold Time**: 클럭 엣지 후 데이터가 안정적으로 유지되어야 하는 최소 시간입니다.
- **CDC**: Clock Domain Crossing. 서로 다른 clock domain 사이의 신호 전달 문제입니다.

## Analog

- **Operating Point**: 회로의 DC bias 상태입니다.
- **gm**: Transconductance. 입력 전압 변화에 대한 출력 전류 변화율입니다.
- **ro**: Output resistance. channel length modulation 등으로 생기는 소신호 출력 저항입니다.
- **Phase Margin**: feedback system의 안정성 지표입니다.
- **Slew Rate**: 출력 전압이 시간에 따라 변할 수 있는 최대 기울기입니다.
