# 1. Amplifier Basics

Amplifier는 입력 신호를 출력 신호로 변환하는 two-port 회로이다. 중요한 것은 gain 값만이 아니라, 입력 source를 얼마나 loading하는지, 출력 load를 얼마나 잘 구동하는지, 그리고 bias point 주변에서 얼마나 선형적으로 동작하는지이다.
