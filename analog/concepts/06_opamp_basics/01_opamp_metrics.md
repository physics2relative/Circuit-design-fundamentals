# 6-1. Op-Amp Metrics

Op-amp 성능은 하나의 숫자로 결정되지 않는다. 대표 metric은 다음이다.

```text
DC gain
unity-gain bandwidth
phase margin
slew rate
settling time
input offset
input-referred noise
input common-mode range
output swing
CMRR
PSRR
power consumption
load capacitance driving capability
```

Spec은 서로 trade-off 관계가 있으므로 목표 application에 맞게 우선순위를 정해야 한다.
