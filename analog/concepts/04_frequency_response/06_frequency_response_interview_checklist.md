# 4-6. Frequency Response Interview Checklist

- Pole과 zero는 transfer function에서 어떤 의미인가?
- Bode plot에서 pole 이후 magnitude slope와 phase는 어떻게 변하는가?
- -3 dB bandwidth는 무엇인가?
- Gain-bandwidth product는 어떤 조건에서 유용한 근사인가?
- Miller effect는 왜 input capacitance를 키우는가?
- Common-source amplifier에서 `Cgd`가 왜 bandwidth를 줄이는가?
- Load capacitance가 커지면 bandwidth와 stability가 어떻게 변하는가?
- AC simulation과 transient step response는 어떻게 연결되는가?
