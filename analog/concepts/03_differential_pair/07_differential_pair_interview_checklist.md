# 3-7. Differential Pair Interview Checklist

- Differential pair의 tail current는 입력에 따라 어떻게 나뉘는가?
- Differential input과 common-mode input의 차이는 무엇인가?
- Differential pair를 transconductance stage로 볼 수 있는 이유는 무엇인가?
- CMRR은 무엇이고 왜 중요한가?
- Tail current source의 output resistance가 CMRR에 미치는 영향은 무엇인가?
- Input common-mode range는 왜 제한되는가?
- Active load는 왜 gain을 키우는가?
- Differential-to-single-ended conversion은 어떻게 이루어지는가?
- Offset의 주요 원인은 무엇인가?
- Differential pair에서 어떤 simulation을 수행해야 하는가?
