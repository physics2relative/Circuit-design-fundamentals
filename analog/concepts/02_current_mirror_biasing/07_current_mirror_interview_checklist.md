# 2-7. Current Mirror Interview Checklist

- Current mirror가 current를 복사하는 원리는 무엇인가?
- Diode-connected MOS는 어떤 역할을 하는가?
- Current ratio가 W/L ratio로 정해지는 조건은 무엇인가?
- Channel length modulation이 mirror 정확도에 어떤 영향을 주는가?
- Output resistance가 큰 current source가 좋은 이유는 무엇인가?
- Compliance voltage란 무엇인가?
- Cascode mirror는 왜 output resistance를 키우는가?
- Cascode mirror의 단점은 무엇인가?
- BMR은 왜 VDD에 덜 민감한가?
- BMR이 process에는 여전히 민감한 이유는 무엇인가?
- BGR에서 CTAT/PTAT 조합은 무엇인가?
- Process variation은 어떤 방식으로 줄이거나 보정하는가?
