# OP AMP Lab

## Objective

Virtuoso/Spectre 기반 OP AMP 실습 결과를 공개 가능한 노트로 정리합니다.

## Files to Commit

- 설계 의도 설명
- 직접 그린 블록 다이어그램
- 공개 가능한 시뮬레이션 조건
- 주요 파형 캡처 또는 수치 요약

## Files Not to Commit

- PDK/model files
- Cadence library database
- license files
- generated netlists from proprietary environments
