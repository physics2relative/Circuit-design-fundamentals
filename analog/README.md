# Analog Design

Virtuoso + Spectre 기반 실습과 아날로그 회로 기본 개념을 공개 가능한 노트 형태로 정리합니다.

## Topics

1. MOSFET Basics
2. Small-Signal Model
3. Current Mirror
4. Differential Pair
5. OP AMP
6. Frequency Response
7. Virtuoso/Spectre Lab Notes

## Public Note Rule

PDK, 모델 파일, Cadence library 원본, 라이선스 파일은 올리지 않습니다. 회로 개념, 직접 작성한 해석, 공개 가능한 시뮬레이션 조건과 결과 요약만 기록합니다.
