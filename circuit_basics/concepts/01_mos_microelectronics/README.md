# 1. MOS Microelectronics

MOSFET은 gate 전압으로 channel의 전도도를 조절하는 소자이다. 회로설계 관점에서는 MOSFET을 단순한 switch로만 보지 않고, terminal voltage에 따라 current, resistance, capacitance, leakage가 달라지는 비선형 소자로 이해해야 한다.
