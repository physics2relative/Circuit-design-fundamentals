# 5-7. Analog Blocks Interview Checklist

## Small-signal

- small-signal 해석이 무엇인가?
- gm의 의미는 무엇인가?
- ro는 왜 유한한가?
- gmro가 analog gain에서 중요한 이유는 무엇인가?


## Amplifier port model

- Amplifier를 two-port network로 본다는 것은 무슨 의미인가?
- Input resistance가 크다는 것은 source 입장에서 어떤 의미인가?
- Output resistance가 작다는 것은 load driving 관점에서 어떤 의미인가?
- Voltage amplifier의 이상적인 `Rin`, `Rout` 조건은 무엇인가?
- Transconductance amplifier는 어떤 신호 변환을 하는가?
- Transresistance amplifier는 어떤 신호 변환을 하는가?
- Current amplifier의 이상적인 `Rin`, `Rout` 조건은 무엇인가?
- Loading effect 때문에 실제 gain이 줄어드는 과정을 설명할 수 있는가?
- Common-source, source follower, common-gate, current mirror를 amplifier type 관점에서 분류할 수 있는가?

## Amplifier

- common-source amplifier가 inverting인 이유는 무엇인가?
- source follower는 왜 buffer로 쓰이는가?
- common-gate의 input impedance 특성은 어떤가?

## Bias / mirror

- current mirror의 동작 원리를 설명할 수 있는가?
- current mirror 정확도를 제한하는 요인은 무엇인가?
- active load는 왜 gain을 키우는가?

## Differential pair

- differential pair는 tail current를 어떻게 나눠 갖는가?
- common-mode와 differential-mode의 차이는 무엇인가?
- CMRR이 왜 중요한가?
