## 학습 흐름

1. 전압/전류/전력, source, KCL/KVL, divider, RC transient를 회로 해석의 기본 언어로 잡는다.
2. Carrier, drift/diffusion, PN junction, MOS capacitor를 먼저 잡고 MOSFET으로 넘어간다.
3. CMOS inverter에서 noise margin, delay, power가 어떻게 생기는지 본다.
4. VTH, load capacitance, PVT 변화가 timing과 power에 주는 영향을 연결한다.
5. gm, ro, current mirror, differential pair로 analog gain block의 기반을 잡는다.
6. op-amp의 gain, bandwidth, phase margin, slew rate를 system 관점에서 정리한다.
