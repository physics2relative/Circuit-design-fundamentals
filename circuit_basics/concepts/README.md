# Circuit Basics Concepts

기초 회로이론과 CMOS/VLSI 회로 관점의 기본 개념 정리이다. 소자 물리와 analog block 설계는 각각 별도 섹션에서 다룬다.

## 목차

1. [Circuit Theory Basics](./01_circuit_theory_basics/README.md)
2. [CMOS Inverter and VLSI Basics](./02_cmos_inverter_vlsi/README.md)
3. [Delay, Power, and PVT](./03_delay_power_pvt/README.md)

## 학습 흐름

1. 전압/전류/전력, source, KCL/KVL, divider, RC transient를 회로 해석의 기본 언어로 잡는다.
2. CMOS inverter에서 noise margin, delay, power가 어떻게 생기는지 본다.
3. VTH, load capacitance, PVT 변화가 timing과 power에 주는 영향을 연결한다.

- amplifier/op-amp 구조는 `analog/`를 참조한다.
