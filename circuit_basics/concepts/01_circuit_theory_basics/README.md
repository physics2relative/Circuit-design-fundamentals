# 1. Circuit Theory Basics

회로이론 기본은 MOSFET, amplifier, op-amp, reference 회로를 해석하기 위한 공통 언어이다. 이 장에서는 이상 전압원/전류원, 저항, KCL/KVL, voltage/current division, Thevenin/Norton equivalent, RC transient, impedance를 회로설계 면접에서 설명할 수 있는 수준으로 정리한다.

## 학습 방향

- 전압, 전류, 전력의 기준 방향과 부호를 명확히 한다.
- 이상 source와 실제 source의 차이를 output resistance 관점으로 이해한다.
- KCL/KVL을 node/loop 해석의 기본 도구로 사용한다.
- 저항 분배와 loading effect를 input/output resistance와 연결한다.
- RC time constant를 delay, bandwidth, pole의 기초로 본다.
- impedance를 통해 capacitor와 inductor를 주파수 영역에서 해석한다.
