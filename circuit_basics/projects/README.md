# Circuit Basics Projects

Spectre/Virtuoso 기반 기초 회로 실습을 정리하는 공간이다.

## 예정 프로젝트

1. `cmos_inverter_spectre`: CMOS inverter VTC, transient delay, power 측정
2. `common_source_amp_spectre`: common-source amplifier DC operating point, gain, bandwidth 확인
3. `opamp_spectre`: two-stage op-amp gain, phase margin, transient response 확인

대용량 결과 파일은 commit하지 않고, 필요한 경우 파형 캡처 이미지와 핵심 setup만 정리한다.
