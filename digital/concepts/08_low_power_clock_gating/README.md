# Low Power and Clock Gating

## 작성 범위

전력 절감과 clock gating 개념을 정리하는 대단원이다.

## 포함할 주제

- dynamic power
- switching activity
- clock power
- clock enable
- clock gating
- integrated clock gating cell
- latch-based clock gating
- glitch-free clock gating
- clock gating check
- enable timing
- low-power RTL 기본

## 정리 방향

Clock gating은 clock switching을 줄여 dynamic power를 줄이는 기법이다. 일반 RTL의 clock enable과 실제 clock gating cell의 차이를 구분해 정리한다.
