# Verilog Basics

RTL 설계를 위한 Verilog 기본 문법과 코딩 스타일을 정리합니다.

## 핵심 항목

- `wire`와 `reg`
- `assign` statement
- `always @(*)`
- `always @(posedge clk)`
- blocking assignment `=`
- non-blocking assignment `<=`
- `parameter`
- `generate`
- `case`, `if-else`
- testbench 기초
- synthesizable Verilog
