# Timing and STA

## 작성 범위

Timing과 static timing analysis를 정리하는 대단원이다.

## 포함할 주제

- propagation delay
- clock-to-Q delay
- setup time
- hold time
- clock period
- clock skew / jitter
- critical path
- slack
- setup violation
- hold violation
- timing closure
- multicycle path / false path 개념

## 정리 방향

회로가 목표 clock frequency에서 안정적으로 동작 가능한지 판단하는 기준을 정리한다. CDC나 reset release처럼 clock domain과 관련된 내용은 `../07_clock_reset_cdc/`와 연결해 정리한다.
