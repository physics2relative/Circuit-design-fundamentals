# 3. MOS Capacitor

MOS capacitor는 MOSFET의 gate control 원리를 이해하기 위한 기본 구조이다. Gate voltage가 semiconductor surface charge를 바꾸고, accumulation/depletion/inversion 상태를 만든다.
