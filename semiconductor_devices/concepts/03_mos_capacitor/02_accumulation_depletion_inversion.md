# 3-2. Accumulation, Depletion, and Inversion

P-type substrate 기준으로 gate voltage에 따라 surface 상태가 바뀐다.

## Accumulation

Gate에 음전압을 걸면 hole이 surface에 모인다.

```text
negative VG -> hole accumulation
```

## Depletion

Gate 전압을 양의 방향으로 올리면 hole이 밀려나고 fixed acceptor ion이 남아 depletion region이 생긴다.

```text
positive VG -> hole repelled -> depletion
```

## Inversion

Gate 전압을 더 올리면 electron이 surface에 모여 p-type surface가 n-type처럼 동작한다.

```text
strong inversion -> conductive channel 형성 가능
```

NMOS는 이 inversion layer를 source-drain channel로 사용한다.
