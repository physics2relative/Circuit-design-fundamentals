# Semiconductor Devices

반도체 소자 기본 개념을 대학원 면접과 반도체 직무지식 대비 관점으로 정리한다. 이 섹션은 회로이론이나 회로 block 설계가 아니라, **반도체 내부에서 carrier, junction, capacitor, transistor가 어떻게 동작하는가**를 다룬다.

## Structure

```text
semiconductor_devices/
  concepts/   # carrier, PN junction, MOS capacitor, MOSFET, BJT, non-ideality
  assets/     # band diagram, C-V curve, I-V curve 등 직접 그린 그림
```
